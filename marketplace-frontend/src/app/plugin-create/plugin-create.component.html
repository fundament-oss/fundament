Build a plugin

Plugins package platform and application functionality that anyone can install into their clusters. You build one with the functl command-line tool and publish it to the marketplace after a central review.

Before you start

  • Install the functl CLI and make sure it is on your PATH.
  • Have access to a cluster you can sideload onto for testing.
  • A container registry account is not needed — functl pushes to the Fundament registry.

What happens after you push

Your pushed build shows up under My plugins. From there you can sideload it onto a cluster to try it out, then submit it for central review. Once the review team approves it, the plugin is published to the marketplace catalog for everyone to install.
